# Task Management and Productivity Tracker

This project is a Task Management and Productivity Tracker built using Angular and TypeScript. The application allows users to create, manage, and track tasks with priority levels, due dates, and completion status.

## Technologies Used
- Angular
- TypeScript
- Angular Material
- RxJS
- Local Storage

## Features
- Add new tasks
- Edit existing tasks
- Delete tasks
- Mark tasks as completed
- Filter tasks (All, Pending, Completed)
- Highlight tasks based on priority
- Task progress tracker
- Persistent data using local storage

## Application Components
- Task List
- Task Form
- Task Detail
- Task Edit
- Completed Tasks
- Navigation Bar

## UI Framework
Angular Material was used to create a modern and responsive interface.

## Productivity Feature
A progress tracker shows the number of completed tasks and displays a progress bar for task completion.
